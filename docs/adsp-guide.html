모수 vs 비모수 검정
Parametric vs Non-parametric Test
모수 검정
데이터를 자로 재서 비교
실제 숫자(평균·분산)로 판단
비모수 검정
데이터를 줄 세워서 비교
등수·순위로만 판단
💡 두 반 키 비교 비유
모수
한 명씩 cm로 재서 평균 내고 비교
→ "1반 155cm, 2반 160cm" — 실제 숫자 사용
비모수
키 순서대로 줄 세우고 등수 합으로 비교
→ 실제 cm는 안 씀, 순위만 씀
🏀
왜 비모수를 쓰나?
평균은 튀는 값 하나에 잘 속는다.
210cm 농구선수 한 명이 반 평균을 확 올려버림.
등수로 보면 210cm든 165cm든 그냥 "1등"이라 이상치에 안 흔들림.
모수 검정을 쓸 때
✓ 데이터가 많고
✓ 정규분포 (종 모양)
✓ 이상치 없음
→ 더 정확하고 검정력 강함
비모수 검정을 쓸 때
✓ 데이터가 적거나
✓ 분포를 모를 때
✓ 이상치 있음
→ 더 안전, 분포 가정 불필요
구분 모수 검정 비모수 검정
사용 대상 실제 값 (평균·분산) 순위 (등수)
분포 가정 정규분포 가정 ✓ 가정 없음 ✓
데이터 양 많을 때 유리 적어도 OK
이상치 약함 (잘 속음) ⚠️ 강함 (안 속음) ✓
검정력 강함 상대적으로 약함

🎯 시험 포인트
  • 비모수 검정은 정규분포 가정이 불필요하다 — 자주 나오는 선지
  • 이상치에 강한 것은 비모수, 검정력이 강한 것은 모수
  • 스피어만 상관계수 = 비모수 / 피어슨 상관계수 = 모수
  • 크루스칼-왈리스 = 비모수판 ANOVA (세 집단 이상 순위 비교)
